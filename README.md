# Food Forest Informatie Platform (Frontend)

## Overview
Dit project is een Frontend applicatie ontwikkeld om mensen te informeren over voedselbossen, en om voedselbossen te helpen om op te starten. Een voedselbos is een zelf-verzorgend ecologisch systeem dat een volledig natuurgebied nabootst, terwijl het eetbare en nuttige planten voorziet. Veel mensen die graag willen starten met voedselbossen zijn niet bekend met de systemen van een voedselbos en hoe ze met elkaar omgaan.

Dit platform probeert de kenniskloof te dichten door makkelijke, eenvoudige en interactieve elemeten toe te voegen aan het systeem. Deze elementen die demonstreren hoe het voedselbos werkt en wat de status is.

## Doel en Doeleinden
De belangrijkste doeleinden van het platform zijn:
- Het introduceren van het concept voedselbossen aan mensen met weinig of geen voorkennis.
- Het demonstreren hoe duuzaamheid wordt verhoogd door voedselbossen en de systemen hierbinnen.
- Het ondersteunen van gebruikers die een voedselbos willen starten maar nog niet op het punt zitten dat het bos zelfvoorzienend is.
- Het laten zien van informatie op een duidelijke, informatieve en makkelijke manier waarvan je geinteresseerd raakt maar ook wat van leert.

De doelgroep bestaat uit beginnende voedselbossen, studenten en mensen rond die leeftijd die een basis willen van hoe een voedselbos te werk gaat.

## Functionaliteiten
- Svelte-gebaseerde interactieve interface
- Inspelen op een fijne oogvangende website die de systemen binnen een voedselbos aantonen. 
- Informatieve inhoud gericht op verkenning en onderzoeking
- Gestructureerde UI die verkenning stimuleert i.p.v. passief lezen

Hoewel de huidige versie zich richt op presentatie en interactie, gaan wij ook nog met sensoren werken om accurate data van voedselbossen in het systeem te verwerken.

## Tech stack
Deze frontend applicatie is gebouwd met:
- Svelte
- Moderne webdevelopment tooling

## Projectstructuur
Deze repository bevat uitsluitend het frontend gedeelte van het platform.  
Een backend bestaat als een afzonderlijk systeem en zal later worden aangevuld met microservices, al zijn de specifieke services nog niet vastgesteld. Deze keuzes maken wij naarmate wij verder in het project komen.

## Installatie en uitvoeren van het project
Om de frontend lokaal te installeren en uit te voeren:

```bash
npm install
npm run dev
```

De development server zal starten en een lokaal adres beschikbaar maken voor testen.
